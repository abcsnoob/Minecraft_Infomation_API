1. Giới thiệu

Abc's NoobMC Skin Viewer là giải pháp tối ưu cho việc hiển thị skin người chơi Minecraft...

2. Cài đặt thư viện

Thư viện được phân phối qua CDN, không cần cài đặt phức tạp.

<script src="https://cdn.jsdelivr.net/gh/abcsnoob/Minecraft_Infomation_API@main/dist/abcsmc.min.js"></script>

3. Sử dụng Fetch API (Advanced)

Nếu bạn cần tùy biến logic xử lý dữ liệu sâu hơn, hãy sử dụng Fetch:

// Hàm lấy dữ liệu chi tiết
async function fetchPlayerData(username) {
    const endpoint = `https://mcskin.abcsnoob.workers.dev/api/${username}?newinfo=true&cape=true`;
    const response = await fetch(endpoint);
    return await response.json();
}

4. Xử lý Cape trong Viewer 3D

Để đảm bảo Cape hiển thị chính xác, bạn cần kiểm tra query string...

const viewerUrl = data.skin3d + (data.skin3d.includes('?') ? '&' : '?') + 'cape=true';
