Cách hiển thị Cape trong 3D Viewer

Để trình xem 3D hiển thị kèm Cape, bạn cần nối thêm tham số ?cape=true vào URL thuộc tính skin3d nhận được từ API.

// Ví dụ cách render iframe 3D với Cape
const skin3dUrl = data.skin3d + (data.skin3d.includes('?') ? '&' : '?') + 'cape=true';
iframe.src = skin3dUrl;

Tích hợp qua Fetch API

async function getMinecraftInfo(username) {
    const url = `https://mcskin.abcsnoob.workers.dev/api/${username}?newinfo=true&cape=true`;
    const response = await fetch(url);
    const data = await response.json();
    
    // Sử dụng data.skin3d + "?cape=true" để hiển thị kèm Cape
    console.log("Skin 3D với Cape:", `${data.skin3d}?cape=true`);
}
